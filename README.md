# 🌱 Tanami: Smart Urban Farming Solution

![Status](https://img.shields.io/badge/Status-Incubation-green)
![Category](https://img.shields.io/badge/Category-IoT%20%26%20Mobile-blue)

**Tanami** adalah solusi ekosistem pertanian perkotaan berbasis IoT yang menghubungkan perangkat keras cerdas dengan kemudahan aplikasi mobile.

## 🏆 Achievements
* **1st Winner GEMASI - Kategori APSI (2025)**

## 🚀 Key Features
* **Real-time Monitoring:** Pantau kelembapan dan suhu secara akurat.
* **Auto-Watering System:** Penyiraman otomatis berbasis sensor.
* **TanamCare AI:** Panduan cerdas ensiklopedia tanaman (Database-driven).
